Exploratory Data Analysis¶

  • The data set has 21,321 instances and 23 features (input variables), which include the Certificate of Deposit’s maturity term, times it got renewed, age of the cd, federal rate, rate spread, and some categorical grouping of the cd's indicators, etc.
  • The predictor variable (“y”) states the outcome of the CD renewal — whether the respondent would renewal for a deposit 1 (“yes”) or 0 (“no”). Let's start with loading all the third-party libraries (you can also make use of requirements.txt) that we are going to use and the data set itself
